# 🛒 Blinkit Sales Analytics Dashboard

## 📌 Overview

This project presents a Blinkit Sales Analytics Dashboard built using Power BI. It focuses on analyzing sales performance, customer behavior, and product trends to generate actionable insights and support business decision-making.

---

## 📊 Dashboard

| Dashboard                                                    | Details                                                                                                                                                                                                                                                                                                                                                                  |
| ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
|  | **Description:** A single-page interactive dashboard providing a complete overview of sales and performance.<br><br>**Key Metrics:** Total Sales, Total Orders, Revenue, Average Order Value, Category-wise Sales.<br><br>**Insights:** Identifies top-performing products, sales trends, and customer behavior patterns to support business growth and decision-making. |

## 🛠️ Tools & Technologies

## 🎯 Conclusion

This project demonstrates how data analytics can transform raw sales data into meaningful insights. It helps improve business performance, optimize strategies, and enhance customer understanding.
